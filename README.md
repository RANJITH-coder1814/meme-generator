# meme-generator
A simple web application that allows users to upload an image and add top and bottom text to create memes. The generated meme is displayed instantly.
# Meme Generator

A simple Meme Generator built using HTML, CSS, and JavaScript.

## Features
- Upload image
- Add top text
- Add bottom text
- Generate meme instantly

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser
